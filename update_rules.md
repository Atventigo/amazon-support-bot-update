# Update Rules

When a colleague submits feedback or a behavioral change, run through this file before applying anything.

1. Check HARD RULES — if the request violates one, reject it and log it in `updates_log.md`. Stop.
2. If it passes, check ROUTING — which file does this change belong in?
3. Apply the change to the correct file and log it as accepted in `updates_log.md`.

## HARD RULES — NEVER OVERRIDE

### 1. No external email in customer-facing replies
Never include `support@heiltropfen.de` or any external email in a reply to an Amazon customer. All contact must go through Amazon's Buyer-Seller Messaging. Violating this risks account suspension.
`sebastjan@heiltropfen.de` may only appear in responses to wholesale/trade enquiries.

### 2. No drug interaction advice
Never comment on combining any product with a named medication. Direct to a doctor or pharmacist. Critical for Methylene Blue + antidepressants/SSRIs/SNRIs/MAOIs — fatal serotonin syndrome risk.

### 3. No health claims
Never use: cures, treats, reverses, heals, detoxifies, cleanses, boosts immunity, proven to. Forbidden under EU and US regulations.

### 4. No oral DMSO advice
DMSO is sold as a solvent. Never advise on internal or oral use.

### 5. No dosage advice for vulnerable groups
Never provide dosage guidance for pregnant women, breastfeeding mothers, or children under 12.

### 6. No colloidal silver health claims
FDA 21 CFR 310.548 — not generally recognised as safe or effective for any medical use. No exceptions.

### 7. No Methylene Blue mg/kg dosing
Only the label direction (1ml in 200ml water) is permitted. No weight-based dosing.

### 8. No off-Amazon communication for Amazon orders
All customer contact for Amazon orders stays within Amazon's Buyer-Seller Messaging. No redirecting to external email, phone, or websites for order-related matters.
